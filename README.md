# aprendendo-python

Este repositório contém meus primeiros exercícios em Python e serve como meu diário de bordo na programação.

## 🚀 Projetos e Exercícios Iniciais
* **conta_e_gorjeta.py:** Um script simples para calcular faturamento adicionando a taxa de serviço/gorjeta. Bom para praticar operadores matemáticos e manipulação de tipos (float/int).
* **automatizando_loja.py:** Primeiros passos simulando a lógica de automação de processos comerciais.

## 📚 Tecnologias e Conceitos Estudados
* Variáveis e tipos de dados (Strings, Inteiros, Floats)
* Operações matemáticas básicas
* Entrada e saída de dados (`input` e `print`)
