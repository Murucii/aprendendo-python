# 📦 Criando Executáveis com PyInstaller

Nesta pasta, guardo meus testes e o aprendizado sobre como transformar scripts Python (`.py`) em arquivos executáveis independentes que rodam direto no computador.

---

## 🚀 Como Funciona (Passo a Passo)

### 1. Instalar a ferramenta
No terminal do seu computador, instale o PyInstaller:
```bash
pip install pyinstaller

depois de instalar usar o comando:

pyinstaller --onefile seu_script.py

e procurar o arquivo na sua pasta dist.
